## 常见问题与解答

**Q: 改动后网页无法加载怎么办？**
A: 清除浏览器缓存（Cmd+Shift+R 在 Mac 上），或打开开发者工具（F12）检查控制台是否有 JavaScript 错误。

**Q: renderProfile() 很长，是否会影响性能？**
A: 不会。这只是 HTML 字符串的生成，每次加载页面才执行一次。

**Q: 为何输入框内容无法实时更新预览？**
A: 这是可选功能，需要额外编程。目前刷新页面后预览会更新。如需实时更新，可参考上面的"实时预览"建议。

**Q: 为什么手机上的布局有时不对？**
A: 清除缓存并硬刷新（Cmd+Shift+R），确保加载的是最新的 CSS 文件。
